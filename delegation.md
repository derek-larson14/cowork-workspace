# Delegation

Claude's task queue. Add items here, then run `/delegate` to work through them.

This is the stuff you want Claude to take on: research, data analysis, file organization, drafting, setup, prototyping. Anything that does not need your voice, your relationships, or your judgment.

When Claude finishes a task, it checks the box and leaves a one-line note about what was done and where the output lives.
